# Workflow-Management-System

## Objectives
  - To be able to create a system which would help to manage a security company in its day to day business.
  - Manage the customers
  - Manage the employees


## Requirements
  - Should be able to add building information
  - Should be able to add new employees
  - Should be able to assign an employee to an building
  - Show employee schedule


## Usecases
- Happy Path: Assign an employee to a building duty.
- Non-Happy Path: Unable to assign an employee to a building duty.
